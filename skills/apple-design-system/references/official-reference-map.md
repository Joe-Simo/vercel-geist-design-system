## Drift Rules

- If an expected Apple page returns 404, do not add or cite it. Search current
  official Apple docs, Apple Developer App Store/marketing pages, or Apple legal
  pages as appropriate, then cite the resolved official page.
- If a listed page redirects, record the final canonical Apple URL in Docs
  Evidence and update the local map when editing the skill.
- Archived Apple docs and old PDFs are fallback context only when no current
  Apple Developer page exists for the topic.
- Third-party UI kits, Figma community files, blog posts, screenshots, and
  cloned app screens never satisfy Docs Evidence.
